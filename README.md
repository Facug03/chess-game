## Correr localmente

Clona el proyecto

```bash
  git clone https://github.com/Facug03/chess-game.git
```

Movete al directorio del proyecto

```bash
  cd chess-game
```

Instala las dependencias

```bash
  npm install
```

Inicia el proyecto

```bash
  npm run dev
```
